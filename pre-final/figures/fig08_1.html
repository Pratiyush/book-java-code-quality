The principle is kept; the idiom is updated. Three verdicts: Stands, Served by a feature, Reinforced-and-dated.

Legend Stands principle + idiom both current Served by a feature principle stands; language satisfies the common idiom Reinforced-and-dated principle and direction hold; tool set expanded
Canon-dating discipline: a book is a secondary authority. Where a modern JEP or the JLS changes the terrain, the primary source wins and the EJ item is contextualized, not presented as current fact without primary confirmation. Preview/exploratory features (structured concurrency, Valhalla value classes) are AHEAD-OF-PIN — never cited as stable replacements.

EJ principles: Effective Java, Joshua Bloch, 3rd ed. 2018 (Items cited above). JEP sources pinned at JDK 21.0.11 / 25.0.3: JEP 395 (records), JEP 409 (sealed types), JEP 441 (pattern matching for switch), JEP 444 (virtual threads); JLS §4 (generics), JLS §8.9 (enum).
