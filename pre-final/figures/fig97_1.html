  • Coverage theater — assertion-light tests inflate coverage metric, verified by mutation score

AI refactoring: not behavior-preserving by construction (unlike OpenRewrite/Refaster, Ch 40) — requires a green test net before merging. Test generation critical guardrail: generate from the spec, never from the implementation; verify tests fail on broken code via mutation testing (Ch 23). Statistics on AI code defect rates are dated snapshots — verify and attribute to the specific study; the structural gate discipline is durable where any percentage is not.
