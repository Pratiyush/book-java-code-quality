# Scoring packets — paste-and-go

One file per chapter, named `ch-<printed#>_<slug>.md`, in FINAL_INDEX reading order. Each is fully
self-contained: the independent-review prompt **plus** the chapter draft, ready to paste into a
different-vendor LLM in one shot.

## Use

1. Open `ch-NN_<slug>.md`, copy **everything below the `=====` header line**, paste into a fresh chat
   in a **different-vendor LLM** (ChatGPT, Gemini, etc. — not Claude; the independence is the gate).
2. It returns a one-pager scorecard. Save it **verbatim** at the path printed at the top of the packet:
   `03-drafts/<slug>/<slug>_SCORE_INDEP.md`.
3. `python3 .claude/scripts/status.py` then auto-promotes any chapter scoring **≥88% (44/50)** with
   floors A/B/C-source PASS into `04-approved/`. Below 88% → Claude lifts it and you re-score.

## Order

Score in the order in `../QUEUE.md`: **Batch A first** (the chapters that came through code-review +
the gate dry-runs clean — most likely to clear 88% immediately), then **Batch B** (each carries a named
lift target Claude is addressing). The 8 chapters already scored by Sonnet (printed 38–47 cluster) are
worth a fresh different-vendor pass — it replaces the Sonnet score.

Packets are regenerated by the H1 step; if a draft changes after a lift, regenerate that packet before
re-scoring so the reviewer sees the lifted text.
