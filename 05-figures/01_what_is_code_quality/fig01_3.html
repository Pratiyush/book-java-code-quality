Prudent — a tool: debt tracked and/or repaid as understanding grows
Reckless — damage: no plan, no visibility, interest accrues silently
Ward Cunningham's original metaphor (OOPSLA 1992) was prudent and inadvertent — shipping first-time code and refactoring as knowledge grows. Fowler's quadrant (TechnicalDebt bliki) clarifies that reckless debt is not a "debt" in the useful sense: it is damage with a respectable name. This book holds the line: prudent, tracked debt is a tool; reckless debt is not.
