# fig50_1.sources.md — source-trace sidecar for Fig 50.1

**Chapter:** 24 — Contract & Approval Testing (dossier key 50, folds 52)
**Figure:** Fig 50.1 — Pact four-stage pipeline
**Family:** Flow-sequence
**One idea:** The Pact consumer-driven contract guarantee is only delivered when all four stages run in sequence: consumer test generates the pact, the broker publishes it, provider verification replays it against the real provider, and can-i-deploy gates deployment on the broker's consumer×provider verification matrix.
