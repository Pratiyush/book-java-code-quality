## Accessibility

- Alt-text: Canon-dating table mapping seven Effective Java principles to the modern Java feature that updated each idiom, with a Stands / Served / Reinforced verdict.
- Long-description: A table with three columns — Effective Java principle (3rd ed., 2018), modern Java feature with its JEP and version, and a verdict — across seven rows. Row 1: minimize mutability (Items 15–17) maps to records, a transparent final-field data carrier (JEP 395, final in Java 16), verdict Served by a feature, noting the common transparent-data case is one line and to hand-write when invariants, identity, or hidden representation are needed. Row 2: obey the equals / hashCode / toString contracts (Items 10–12) maps to records generating all three from components (JEP 395), dual verdict Served for data carriers and Stands for identity-bearing classes. Row 3: model a closed set of types safely (Items 23, 41) maps to sealed types plus pattern matching for switch (JEP 409 sealed in Java 17, JEP 441 pattern switch in Java 21), verdict Served / Reinforced, an exhaustive compiler-checked alternative to the instanceof ladder. Row 4: prefer enums for fixed instances and singletons (Items 3, 34) maps to enum, unchanged (JLS 8.9), verdict Stands, no modern feature displaces it. Row 5: use generics and avoid raw types (Items 26–33) maps to JLS chapter 4 generics, unchanged since Java 5, verdict Stands, deep dive in Chapter 11. Row 6: prefer alternatives to Java serialization (Item 85) has no JEP retiring the advice, verdict Stands, deep dive in Chapter 30. Row 7: prefer concurrency utilities over wait/notify (Item 81) maps to virtual threads (JEP 444, Java 21), with structured concurrency noted as preview and AHEAD-OF-PIN, verdict Reinforced-and-dated, Chapter 14. A legend defines the three verdicts: Stands means principle and idiom both current; Served by a feature means the principle stands and the language satisfies the common idiom; Reinforced-and-dated means the principle and direction hold while the tool set expanded. A closing note states the canon-dating discipline — a book is a secondary authority, so where a modern JEP or the JLS changes the terrain the primary source wins, and preview or exploratory features (structured concurrency, Valhalla value classes) are AHEAD-OF-PIN and never cited as stable replacements.
- Grayscale-safe: Yes. Each verdict badge carries its full text label (Stands / Served / Reinforced-and-dated), and the legend maps every label to its meaning, so the verdict is read from the words, not from the badge fill or outline color. Reads in black-and-white print.
- Contrast: Sufficient. Dark ink on white and lightly-tinted cells; the verdict-badge text is dark enough to read in grayscale.
- Code legibility: The figure carries real authored monospace text for the inline identifiers, JEP numbers, and JLS references (for example equals, hashCode, ObjectInputFilter, JEP 444, JLS 8.9). All are selectable text, not a screenshot, and understandable without syntax color.
