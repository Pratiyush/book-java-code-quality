Source analysis

Reads .java source (AST). Catches style, naming, simple patterns.

Checkstyle · PMD · Spotless
Compile-integrated

Runs inside javac as a plugin. Fastest feedback; can auto-fix.

Error Prone · NullAway · Checker Framework
Bytecode analysis

Reads compiled .class files. Sees what the compiler produced — catches what source hides.

SpotBugs (+ FindSecBugs)
Tool vs. Platform

Analyzers run one check; a platform (SonarQube) aggregates many and tracks findings over time.

SonarQube · SonarLint
Lint vs. Test

Static analysis reasons without running code; tests run it. Complementary — not substitutes.

JaCoCo · PITest · ArchUnit
Shift-left (Ch 4): the same class of check run earlier costs less to fix. Format violations found at the IDE are free; the same violation escalated to a PR comment costs a review cycle. Design the stack so cheap checks run early, expensive ones gate CI.
