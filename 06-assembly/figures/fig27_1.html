Axis rule: A tool can only catch what its vantage point lets it see — the four are complementary by construction. Checkstyle, reading one file with no type information, cannot know whether a custom exception extends Exception (its own docs state so). Error Prone, on the type-attributed tree, knows every static type. SpotBugs, on bytecode, sees the impossible cast javac compiled away into a checkcast. None stands "smarter" than the others — each stands somewhere different.
