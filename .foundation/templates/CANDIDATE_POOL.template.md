## Verification-depth rubric (depth, not word count)

> Dossier depth is measured by **what was inspected and traced**, never by word count. A dossier is complete when every claim it makes is sourced and every {{EXAMPLE_POLICY}} artifact / snippet is verified against the pin. Set the minimum inspection surface per depth band for THIS book's authority type (e.g. for a code book: which guide pages + which source modules to read; for a literature book: which papers + which primary tables).

| Depth band | What to inspect at {{AUTHORITY_PIN}} | What gets traced |
|---|---|---|
| **Foundational** | _(the broadest inspection surface for a rich/foundational key)_ | Every claim / snippet traced to an exact source path at the pin. |
| **Standard** | _(the primary surface + directly linked sources)_ | Every claim / snippet traced to an exact source path. |
| **Conceptual / niche** | _(the closest source; confirm it EXISTS at the pin before asserting)_ | Every claim / snippet traced; mark anything absent at the pin as UNVERIFIED → `09-flags/`. |

**Rules that apply at every band (HARD):**

1. **Trace to the pinned authority.** Every {{INVENT_UNITS}} traces to {{AUTHORITY_SOURCE}} at {{AUTHORITY_PIN}}. NEVER to an off-pin / moving source. Record the exact source path.
2. **Bounded, verified excerpts.** Each snippet/quote stays within the book's ceiling and is verified against the pinned source. Record the source path beside it. _(Snippet line-ceiling is a technical-profile detail — see BOOK-TYPE-PROFILES.md; set the bound your `{{EXAMPLE_POLICY}}` defines.)_
3. **Confirm before asserting.** If a source/section does not exist at the pin, the fact does not exist for this book — cut it or mark it UNVERIFIED and flag to `09-flags/`. Never invent {{INVENT_UNITS}}.
4. **Citation source ranking** (house style): set the ranking for this book's authority type in `00-strategy/GUIDELINES.md` (pinned authority first, down to color-only sources last). Never content farms or AI text as a factual source.
5. **Neutrality on agnostic-sensitive keys.** For any `⚠` key, the dossier binds to the never-crown rule ({{NEUTRALITY_STANCE}}): balanced cases, cited cross-subject claims, zero banned phrasings.
